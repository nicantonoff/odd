# Requisitos Funcionais
RF-001 O Schola deve redirecionar para login e cadastro usando Certificare e receber o retorno autenticado.
RF-002 O Schola deve listar turmas do curso de ProdOps que estão abertas para matrícula.
RF-003 O Schola deve permitir que um usuário autenticado solicite matrícula em uma turma.
RF-004 O Schola deve verificar a disponibilidade da turma e confirmar a matrícula após recebimento de confirmação de pagamento.
RF-005 O Schola deve armazenar os dados dos usuários de forma segura.
RF-006 O Schola deve fornecer logs e métricas para monitorar e diagnosticar problemas.
